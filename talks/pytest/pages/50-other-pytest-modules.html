Catch warnings as errors in test suite¶

-Werror will should cause any tests that have warning errors for deprecation messages to cause the test suite to error

# content of test_show_warnings.py
import warnings

def api_v1():
    warnings.warn(UserWarning("api v1, should use functions from v2"))
    return 1

def test_one():
    assert api_v1() == 1

bash-5.0$ pytest -k warnings -Werror| cat
